# Restaurant Reservation App

Una aplicación web para gestionar reservas de restaurantes.

## Características
- Reservas de mesas por fecha y hora
- Gestión de capacidad (2, 3, 4+ personas)
- Frontend en Preact con Tailwind CSS
- Backend con Node.js/Express
- Base de datos PostgreSQL
- Gestión de estado con Zustand

## Tecnologías
- Preact
- Tailwind CSS
- Zustand
- Node.js
- Express
- PostgreSQL
